Tags / Elementos

  • < !--(Comentário em HTML)-- >

    A tag de coméntário é usada para inserir comentários no seu código e não são exibidos nos browsers.

  • < !DOCTYPE >

    DOCTYPE não é uma tag de HTML, é uma tag usada para indicar ao browser que tipo de documento que irá ser lido, nesse exemplo HTML5.

  • < a >

    Define um hiperlink, usado para linkar uma página para outra.

      Atributos:
    • href

      Exemplo: href="https://github.com/luizfcneto"

    • target
        valores dos atributos de target :
      • _blank = abre o link clicado em uma aba ou janela diferente.
      • _parent = abre o link clicado em uma janela com mesmo parentesco.
      • _self = abre o link clicado na aba atual.
      • _top = abre o link clicado em todo corpo da janela.

  • < h1 > ~ < h6>

  • < br >

    Pula a linha.

  • < hr >

    Quebra de seção

  • < pre >

    Segue a definição pre formatada do texto (que está exatamente escrita no código).

    Entrando em desuso, por não ter controle de qual display o usuário irá utilizar, a pre formatação pode deixar a página com barra lateral.

  • < blockquote >

    Essa tag especifica que a seção(contida nela) é uma citação de alguém ou algum lugar

      • Atributos da tag(elemento) blockquote :
    • cite

      Exemplo: cite="href="http://www.citador.pt/frases/citacoes/temas"

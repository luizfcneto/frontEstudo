instalando a dependencia redux-promise: middleware

    npm i --save-dev redux-promise@0.5.3
    promise permite que sempre que uma action disparar uma promise, ele irá aguardar
    que ela retorne para então retornar o conteudo da action.

    
Importando a dependencia do redux-multi: outro middleware
    
    npm i --save-dev redux-multi@0.1.12
    
    O multi permite que retornemos em uma actionCreator um array com varias actions
    mas essas actions são executadas em paralelas. 

Importando a dependencia do redux-thunk: outro middleware

    npm i --save-dev redux-thunk@2.2.0

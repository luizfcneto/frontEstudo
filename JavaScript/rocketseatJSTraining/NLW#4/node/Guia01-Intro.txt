API ( Application Programming Interface ): 
    Não é uma linguagem;
    Não é uma biblioteca que baixamos, instalamos e configuramos na nossa máquina;

    É um conjunto de especificações e regras definidas dentro da nossa aplicação para fornecer funcionalidades para aplicações externas desde que seja respeitada essas regras e o formato de utilização das funcionalidades. 

    A Api irá retornar uma resposta de acordo com a funcionalidade solicitada juntamente com seus parametros. ( regras definidas e especificadas no desenvolvimento da API ). Visualizar imagem aula01-img00.png

    É reutilizavel para outras aplicações, tanto para mobile quando para aplicações web
